# LOGIN_SCREEN

A modern and responsive Login Screen built with React Native and TypeScript.

This project focuses on creating a clean UI with proper asset management, custom fonts, and reusable components. 
It includes a responsive layout that works well on both Android and iOS devices.

## Features
- Beautiful and minimal login UI
- Custom Roboto fonts
- Image assets integration
- TypeScript for better code quality
- Responsive design for all screen sizes

## Tech Stack
React Native, TypeScript, Expo

## How to Run
1. `npm install`
2. `npx expo start`
